## 4. ALGORITHM UPDATES (A10/A11)

### The A10 Algorithm Shift (Rolled Out May 2025)

Amazon quietly replaced the A9 system with A10 in mid-2025. Critical changes affecting book visibility:

### Key Changes Impacting Non-Fiction Books:

**1. Keyword Stuffing = Penalty (Not Just Ineffective)**
- Amazon now **deprioritizes** descriptions/subtitles with awkward keyword packing
- Natural, conversational language is rewarded
- Robot-sounding copy hurts visibility

**2. Natural Language Processing (NLP) Priority**
- Semantic search now matches reader intent, not just keywords
- Context and relevance matter more than exact keyword matches
- Write descriptions for readers, not algorithms

**3. Off-Amazon Signals Now Count**
- Author websites, social media, newsletters contribute to legitimacy
- External traffic sources boost credibility signals
- Platform diversification is now a ranking factor

**4. Amazon Ads Integration**
- Ad performance now influences organic ranking more directly
- "Quality conversions" weighted higher than click volume
- Low return rates and positive post-purchase behavior matter

**5. Reader Engagement Metrics**
- Session duration on book page
- Bounce rate (readers leaving quickly)
- Read-through rates (for KU titles)
- Review velocity and quality

### A10 Optimization Checklist for "The Divided Brain":

- [ ] **Subtitle**: Natural language, not keyword-stuffed
- [ ] **Description**: Conversational tone, 150-200 words, skimmable format
- [ ] **Author Bio**: Include credentials and human element
- [ ] **Categories**: Precise match to actual content (relevance > reach)
- [ ] **External Presence**: Active author website/social signals

### 2026 Algorithm Trends (A11 Preview):

- **AI-Enhanced Matching**: Amazon testing COSMO system for intent-based recommendations
- **Paid + Organic Integration**: Ad conversion rates increasingly feed organic rank
- **Quality Over Quantity**: Fewer, higher-engagement sales > volume spikes
